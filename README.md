接下来我想在用户上传文件时，实现文件格式检查：
1.不需要检查文件后缀名，因为用户可能会把后缀名改为.fea之类的。所以不管后缀名是什么，我们都先尝试把文件当成表格读取
2.不需要文件大小限制，这个之后我再看情况决定做不做。因为目前只打算给实验室的那几个人使用
3.确保数据完整性，即检查是否有空值、除第一行、第一列之外是否全是数字、第一行、第一列中是否有值是重复的
如果不合规，那么明确告知用户原因。（不要试图帮用户“自动填充平均值”或者“自动删除空行”，因为在科研领域，这种隐性的修改会导致实验结果不可复现。直接拒绝并明确告知错误原因，让研究者自己去清洗数据，是既负责任又简单的做法。）

这个平台是干什么的？
我们这个平台主要受众是研究者，他们的论文里经常要比较各种算法，所以就可以来我们这个平台，使用自己的输入数据，选择并比较各种经典算法，得到详细的结果，然后再手动测一次自己的算法就行了。
也就是说我们这个平台作用是减少研究者的工作量，省去了搭建对照组的时间。

哈哈在这之前我一直以为我们这个平台服务的是普通一线医生...
